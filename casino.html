Projet Casino



Lors de la première année, nous avons vu les bases du développement web avec le HTML et le CSS, puis notre professeur nous a fait découvrir JavaScript un langage qui permet de rendre plus interactives les différentes pages web.

Quand nous avons commencé à maîtriser les principes de base du JavaScript nous avons du effectuer un projet de groupe qui consistait en la création de mini-jeux.

Les trois jeux demandés étaient : une machine à sous, un memory et un rapido.

Menu




Machine à sous

Pour le jeu de la machine à sous, il s’agit d'une librairie qui permet de faire défiler les différentes images. L'objectif du jeu est de miser une somme d'argent, de lancer la machine et d'espérer obtenir 3 images identiques. Si c'est le cas le joueur double sa mise, dans l'autre cas, c'est perdu.

Machine à sous




Memory

Memory, le but du jeu est de retrouver toutes les paires de cartes en un nombre d'essais le plus petit, il y a différentes difficultés de niveau, débutant avec 6 paires à trouver, confirmé avec 8 paires et expert avec 10 paires. Vous pouvez aussi affronter un ami.

Memory




Rapido

Vous devez sélectionner 8 nombres + 1 nombre bonus lorsque vous lancerez votre partie l'ordinateur sélectionnera à son tour 8 nombres + 1 bonus. Selon votre quantité de nombres en commun, votre mise sera multipliée par un coefficient de plus en plus élevé plus vous avez de nombres en commun.
